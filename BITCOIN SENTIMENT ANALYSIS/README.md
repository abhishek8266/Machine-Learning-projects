# 📈 Bitcoin Trading Sentiment Analysis Dashboard

An end-to-end Data Analytics project that explores the relationship between cryptocurrency market sentiment and trading performance.

This project combines Bitcoin Fear & Greed Index data with historical trading records to analyze how market sentiment impacts trading behavior.

---

## 🚀 Project Overview

The goal of this project is to understand whether different market sentiments such as Fear, Greed, Extreme Fear, and Extreme Greed influence trading outcomes.

The project includes data preprocessing, feature engineering, exploratory data analysis, and an interactive Streamlit dashboard.

---

## ✨ Features

- Data Cleaning and Preprocessing
- Timestamp-based Feature Engineering
- Fear & Greed Index Analysis
- Trading Performance Analysis
- PnL Analysis
- Win Rate Analysis
- Trade Volume Analysis
- Interactive Dashboard

---

## 📊 Analysis Performed

- Analyzed 200K+ cryptocurrency trading records
- Studied relationship between market sentiment and trading performance
- Compared average PnL across different sentiment categories
- Analyzed winning and losing trades
- Visualized market behavior patterns

---

## 🖥️ Dashboard Features

The Streamlit application provides:

- Total Trades
- Average PnL
- Win Rate
- Average Trade Size
- Sentiment-wise Analysis Charts
- Trading Performance Visualization
