# Machine Learning Projects

This repository contains Machine Learning and Data Analysis projects developed using Python, Pandas, NumPy, Matplotlib, Seaborn, and Scikit-learn.

## Projects Included

### 1. House Price Prediction using Machine Learning

#### Overview

A machine learning project that predicts house prices using the Ames Housing Dataset. The project includes data cleaning, exploratory data analysis (EDA), feature engineering, and model building using Linear Regression.

#### Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-learn

#### Key Tasks

* Data Cleaning and Preprocessing
* Exploratory Data Analysis (EDA)
* Missing Value Analysis
* Feature Engineering
* Linear Regression Model Development
* Model Evaluation using R² Score

#### Outcome

* Built a predictive model for house price estimation.
* Achieved an R² Score of approximately 0.76.

### 2. Bitcoin Sentiment Trading Analysis

#### Overview

This project analyzes the relationship between cryptocurrency market sentiment and trading activity using the Fear & Greed Index and historical trading data.

#### Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn

#### Key Tasks

* Data Cleaning and Preprocessing
* Sentiment Analysis
* Trading Data Analysis
* Date and Time Feature Engineering
* Data Visualization
* Market Behavior Analysis

#### Outcome

* Analyzed 200K+ trading records.
* Explored the impact of market sentiment on trading behavior.
* Generated insights from sentiment and trading datasets.
