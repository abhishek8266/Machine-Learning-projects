## 📌 Projects Included

# 1. 🏠 House Price Prediction using Machine Learning

## Overview

An end-to-end Machine Learning project that predicts house prices using the Ames Housing Dataset.

The project covers complete ML workflow including data cleaning, exploratory data analysis, feature selection, model training, evaluation, and deployment as an interactive web application.

## Features

- Data Cleaning & Preprocessing
- Missing Value Analysis
- Exploratory Data Analysis (EDA)
- Feature Selection
- Machine Learning Model Building
- Model Evaluation
- Streamlit Web Application Deployment

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Streamlit

## Machine Learning Model

- Algorithm: Linear Regression

## Model Performance

- R² Score: ~0.76

# 2. 📈 Bitcoin Sentiment Trading Analysis

## Overview

A data analysis project that explores the relationship between cryptocurrency market sentiment and trading activity.

The project uses Fear & Greed Index data with historical trading records to understand sentiment-driven market behavior.

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Key Tasks

- Data Cleaning
- Data Preprocessing
- Exploratory Data Analysis
- Time-based Feature Engineering
- Sentiment Analysis
- Data Visualization

## Outcome

- Analyzed 200K+ trading records
- Studied relationship between market sentiment and trading activity
- Generated insights from cryptocurrency datasets
